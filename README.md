# ric-edge-zigbee

## env

| name                      | description                                                | default                
| ------------------------- | ---------------------------------------------------------- | --------------------------------- 
| RIC_EDGE_URL              | URL of RIC Edge core controller (WebSocket)                | ws://localhost:17080/zigbee                              
| ZIGBEE_SERIAL_PATH        | System path of CC2531 USB stick                            | /dev/ttyACM0


## run

```sh
# start
> npm start

# start with repl
> npm run repl
```
